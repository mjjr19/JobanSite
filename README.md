# Sheikh Mohamad Joban Website

Official website for Sheikh Mohamad Joban - Scholar, Educator, and Community Leader.

## Features

- **Homepage** - Hero section featuring the book, about preview, featured videos, and stats
- **About Page** - Biography, education, and roles
- **Book Page** - Information about "30-Year Journey of Da'wah in America"
- **Videos Page** - YouTube video library with integration support
- **Contact Page** - Email contact information

## Tech Stack

- React 19
- Vite
- React Router
- Framer Motion
- Sass/SCSS
- Netlify (Hosting)

## Development

```bash
cd website
npm install
npm run dev
```

## Build

```bash
cd website
npm run build
```

## Deployment

The site is automatically deployed to:
- **Netlify:** https://mjoban.netlify.app
- **Custom Domain:** https://sheikhmohamadjoban.com

GitHub Pages deployment is configured via GitHub Actions (see `.github/workflows/deploy.yml`).

## Repository Structure

```
.
├── website/          # Main website source code
│   ├── src/         # React components and pages
│   ├── public/      # Static assets
│   └── dist/        # Build output
└── .github/         # GitHub Actions workflows
```
