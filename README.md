# Sheikh Mohamad Joban Website

A professional website for Sheikh Mohamad Joban, a respected Muslim scholar, educator, and community leader in Washington State. This website serves as a platform to introduce Sheikh Joban, promote his newly translated English book (based on Kiprah Dakwah), and provide access to his YouTube lectures and contact information.

## Features

- **Home Page** - Hero section featuring the book with prominent CTA
- **About Page** - Biography, education, and community roles
- **Book Page** - Detailed book description and ordering information
- **Videos Page** - Embedded YouTube lectures with filtering
- **Contact Page** - Form for inquiries, invitations, and book orders

## Tech Stack

- React + HTML + CSS + JavaScript
- React Bits components for animations
- Google Fonts (Playfair Display, Inter, Scheherazade New)
- Mobile-first responsive design

## Color Palette

- **Deep Navy Blue** (#0F1A2A) - Primary background
- **Gold Accent** (#D4AF37) - Buttons and highlights
- **Cream White** (#F3EDE1) - Section backgrounds
- **Soft Tan** (#C2A382) - Secondary text
- **Charcoal Gray** (#2D2D2D) - Body text

## Getting Started

### Prerequisites

- Node.js and npm installed
- Git installed

### Installation

```bash
# Clone the repository
git clone <repository-url>

# Navigate to the project directory
cd sheikhJoban-website

# Install dependencies
npm install

# Start the development server
npm start
```
